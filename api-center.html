🏛️ 공공 API 서비스 규격 명세

조회 및 호출할 공공 API를 선택하여 상세 인터페이스 엔드포인트와 필수 매개변수 명세를 확인하세요.

보조금24 서비스 목록 조회 API

GET https://api.odcloud.kr/api/15109950/v1/uddi:de123456

대한민국 전 국민을 대상으로 정부 부처와 지방자치단체가 지급하는 다양한 현금성 및 바우처 보조금 혜택 데이터를 통합 조회합니다.

필수 전송 파라미터 (Headers / Query):
  • `serviceKey` : 공공데이터포털 발급 API 인증키 (Encoding)
  • `page` : 조회 페이지 번호 (Default: 1)
  • `perPage` : 페이지당 노출 데이터 개수 (Max: 100)
API LIVE CONSOLE v1.0
API RESPONSE JSON: HTTP 200 OK
